# Chrome Web Store Listing — PrivacyBlur

## Short Description (132 chars max)
Blur sensitive text, images, and videos on any webpage. Two blur modes, keyboard shortcuts, hover reveal. No data leaves your device.

## Detailed Description

PrivacyBlur lets you hide sensitive content on any webpage with a single click or keyboard shortcut. Perfect for screen sharing, presentations, recording demos, or just browsing in public.

TWO BLUR MODES:
- Sticky Mode (default): Draw a rectangle over any area of the page. The blur zone stays in place even if the page content changes — ideal for dashboards, sidebars, and dynamic content.
- Dynamic Mode: Click individual elements to blur them. The blur follows the element wherever the page puts it.

BLUR ALL: Press Alt+Shift+B to instantly blur all text, images, and videos on the page. Categories are configurable — choose which types of content to blur (text, media, forms, tables, structure).

KEYBOARD SHORTCUTS:
- Alt+Shift+B: Toggle blur all
- Alt+Shift+P: Toggle element picker
- Alt+Shift+U: Clear all blur
- All shortcuts are fully customizable.

HOVER REVEAL: Hover over any blurred element to temporarily peek at the content. Move away and the blur reappears instantly. Also supports click-to-reveal mode.

PER-SITE RULES: Create URL-based rules to automatically apply different blur settings on specific websites.

PRIVACY FIRST:
- Zero network requests — PrivacyBlur never contacts any server
- No analytics, telemetry, or tracking
- All data stored locally in your browser
- Open source: https://github.com/ChennakeshavaG/privacyblur

Works on Chrome 88+ and Firefox 109+.
